# nasa-student-launch-rocket
## Rocket Simulation Design (RockSim)

The initial rocket configuration was designed and analyzed using **RockSim**, a simulation tool commonly used in high-power rocketry to evaluate aerodynamic stability and component layout.

The simulation model defines the rocket geometry, structural components, and internal configuration before physical fabrication.

Key parameters from the simulation:

- Total Length: 135 cm  
- Maximum Diameter: 6.68 cm  
- Mass (without motor): 680 g  

The simulation also calculates the **Center of Gravity (CG)** and **Center of Pressure (CP)** to determine flight stability.
